## 安装

```bash
npm install plugin-toast --save
```


## 例子

```javascript
import Toast from 'plugin-toast'
new Toast({
	icon : 'success',
	text : '加载中'
})
```

### Options

* `icon` none | success |  warn
* `text` 文案
